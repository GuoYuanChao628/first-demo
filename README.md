### 快速启动

- 安装项目依赖包

  `yarn install`

- 启动项目

  `yarn run bulid`
  `yarn run start`

- 项目端口

  `localhost:8090`

### 技术栈

项目主要运用**React** 和**Next.js**服务端渲染框架、And-design UI框架开发的多网页应用，

全局数据流的管理用**Redux**来处理（本项目主要体现在购物车右上角的徽章）

### 项目功能

- 帐号的登录和注册功能（注册只支持手机号码注册，验证码默认888888，否者不能注册成功）
- 购物页面的跳转
- 购物物品的选购
- 购物车的选择，价格的预览

### 说明

项目主要练习组件化和全局数据流的管理，页面的渲染

项目内的数据请求主要是本地json格式的死数据
